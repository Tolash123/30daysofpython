## Flow Control - 2

### Iterative Statements

- for loop
- while loop

#### for loop

- To perform activity for every element
Example
> a = 'bolaji' for i in a: print(i)

## while loop

- It executes group of statements based on the conditions.
Example:
>i = 1
 while i<=3:
    print("Hello")
    i += 1

### Nested Loops
Example
> for i in range(3):
    for j in range(2):
        print("Hello Balaji")
